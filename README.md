# 🚀 Beneficiosi Clients Frontend
## Tecnologías
* [Node.js](#item1)
* [React JS](#item2)
* [TailwindCSS](#item3)
## Instalación
```
git clone https://github.com/Tecsoe/beneficiosi-clients-front.git
cd beneficiosi-clients-front
npm install
cp .env-example .env
```
Configurar variables de entorno en el archivo .env
```
npm run start
```
